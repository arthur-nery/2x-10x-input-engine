# 2XI — 2x–10x Input Engine

A meta-prompt engine for turning tasks into 2x–10x faster execution loops through smaller inputs, rapid feedback, deliberate repetition, faster correction, and measurable learning cycles.

---

## What Is This

**2XI — 2x–10x Input Engine** is a meta-prompt designed to transform a task into a faster execution and learning system.

It does not only improve a prompt. It improves the full execution cycle:

- what will be done;
- the smallest useful input;
- how to generate more valid attempts;
- how to reduce feedback time;
- how to learn from errors faster;
- how to create the next better cycle.

The core idea is simple:

```text
Do more valid cycles in less time.
```

---

## Project Objective

The objective of this repository is to document and improve a reusable prompt for accelerated execution, faster feedback, and deliberate practice.

The prompt is useful for tasks such as:

- studying programming;
- creating LinkedIn posts;
- producing YouTube videos;
- testing ideas;
- improving prompts;
- building projects;
- creating more repetitions with faster correction.

---

## How It Works

The 2XI method turns a task into a structured acceleration plan:

1. Identify the task.
2. Reduce the scope.
3. Diagnose the main bottleneck.
4. Create 2x, 5x, and 10x acceleration options.
5. Generate a practical input package.
6. Create a feedback loop.
7. Define success criteria.
8. Add an anti-perfectionism version.
9. Define the next execution cycle.

---

## Core Rules

The prompt should:

- reduce large tasks into smaller executable versions;
- increase useful repetitions;
- speed up feedback;
- make errors observable;
- turn each error into learning;
- avoid empty volume;
- avoid perfectionism;
- always end with a next cycle.

---

## Activation Command

Use:

```text
2XI: [your task]
```

Example:

```text
2XI: Quero transformar meu estudo de Python de hoje em vídeo, post no LinkedIn e commit no GitHub.
```

## Quick Start

Use the prompt file from the `prompt/` folder.

Current version:

```text
prompt/2xi-2x-10x-input-engine-v0.1.md
```

Paste the prompt into ChatGPT or another LLM, then activate it with:

```text
2XI: [your task]
```

The model should return a structured acceleration plan with 2x, 5x, and 10x execution options.
